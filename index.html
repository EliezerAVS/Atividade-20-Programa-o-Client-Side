3) Faça uma breve pesquisa sobre os principais motores JavaScript do mercado. (Exemplo: V8, Chakra, ...):

  • Rhino, gerido pela Fundação Mozilla, open source, desenvolvido em Java;
  • SpiderMonkey, o primeiro motor de JavaScript, que alimentado Netscape Navigator e poderes hoje Firefox;
  • V8 - open source, desenvolvido pelo Google na Dinamarca, parte do Google Chrome e Node.js;
  • JavaScriptCore - open source, comercializado como Nitro e desenvolvido pela Apple para Safari;
  • KJS - motor ECMAScript / JavaScript do KDE originalmente desenvolvido por Harri Porten para o navegador web Konqueror do projeto KDE;
  • Chakra (JScript9), para Internet Explorer;
  • Chakra (JavaScript), para Microsoft Edge;
  • Nashorn, de código aberto como parte do OpenJDK, escrito pela Oracle Java Línguas e Tool Group;
  • Duktape, um motor embutido com foco na portabilidade e um tamanho compacto;
  • Juce, um ++ C estrutura de aplicativo , contém um interpretador embutido personalizado usando parte da sintaxe do JavaScript;
  • JerryScript, é um motor ultra-leve JavaScript para a Internet das Coisas;
  • Jsish, um motor de JavaScript com verificação de tipo modelado após Tcl;
  • Esper, um auto-interpretador JavaScript com foco em execução no modo seguro e introspecção tempo de execução.
